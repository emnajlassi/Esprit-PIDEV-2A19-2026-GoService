# Nom du Projet
GoService – Plateforme de Services
## Description du Projet 
GoService est une plateforme web développée dans le cadre du programme PIDEV à l’Esprit School of Engineering (2025–2026).  
Elle permet de connecter les utilisateurs avec des prestataires de services tels que plombiers, architectes, avocats et chefs.  

L’objectif de cette plateforme est de simplifier la recherche, la publication de demandes et la réservation de services en ligne, tout en améliorant la communication et l’expérience utilisateur.
